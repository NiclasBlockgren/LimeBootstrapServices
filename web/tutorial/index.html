What is LIME Bootstrap (LBS)?

LBS is:

  • A way of building nice looking HTML things i LIME Pro without worrying to much
  • A framework of smart libraries
  • A open source project, free to contribute to
  • A platform and store for delivering apps
  • A version control system

A Model-View-View Model framework

Heart and soul of LBS:

  • KnockoutJS: The powerful engine behind LBS. Makes smart web GUIs a walk in the park.
  • Twitter Bootstrap: A fantastic web-framework which LBS adapt and extends. Go here for finding out everything about the elements you'll like to use, such as inputs or buttons.
  • Font Awesome: Hundreds of icons ready to use. Available in any size or color as it's really a font!

    • Included smart tools:

  • Moment.js: Makes working with time and date a joy!
  • Underscore.js:Powerful and smart library which takes away the pain when working with data in JavaScript
  • jQuery.js:The defacto standard for working with the DOM. Seldom used in LBS, but there are occasions where jQuery is needed
